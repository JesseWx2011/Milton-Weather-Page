How to Embed

To embed the weather widget on your website, follow these steps:

  1. Copy the following HTML code:
<iframe src="https://jessewx2011.github.io/Milton-Weather-Page/widget.html" width="400" height="300" frameborder="0" title="Weather Widget"></iframe>
  1. Paste the code into your website's HTML where you want the widget to appear.
  2. Adjust the width and height values as needed to fit your website's layout.

Customization Options

You can customize the widget by modifying the CSS in the widget.html file. The widget is responsive and will adapt to different screen sizes.

Download Widget Files

To use the widget on your own website, you'll need to download and host these files:

  • widget.html - The main widget file
  • widget.js - The JavaScript file that fetches weather data
